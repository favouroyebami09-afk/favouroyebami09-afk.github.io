<h1> Introduction to Git Branches </h1>

```console
git branch <name-of-the-branch> # creates a new branch
git checkout -b <name-of-the-branch> # creates a new branch and immediately switch to it 
git checkout <name-of-the-branch> # switch to an existing branch
git branch -d <name-of-the-branch> # delete a branch
git branch # list all branches 
```
