<h1> Vi & Vim </h1>

```console 
1. How to delete a whole line 
    -switch to command mode 
    -type dd

2. Delete 10 lines
    -swith to command mode 
    -type d10 
    -then d again

3. Undo a change 
    -command mode
    -type u 

4. Jump to end of line and switch to insert mode 
    -command mode 
    -SHIFT + A or capital A

5. Jump to end of line w/o switching to insert mode 
    -command mode
    -type $

6. Jump to start of the line 
    -command mode 
    -type 0

7. Go to a specific line number 
    -command line 
    -type the line number plus capital G so
        -type 12G

8. Searching for a pattern 
    -type /pattern Enter 
    -type n to jump to next match
    -type N to search in opposite direction

9. Replace old string with a new one throughout the file 
    -type :%s/old_string/new_string 
    -Enter
```
