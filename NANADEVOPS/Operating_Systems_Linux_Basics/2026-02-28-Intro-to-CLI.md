<h1> Basic Linux Commands </h1>

```console 
$: represent a regular user on an operating system 

#: represent a root user 

cd .. > go up a directory 

cd ../.. > go up two directory 

rm -r [directoryname] > deletes a non empty directory and all the files within it 

rm -d [directoryname] > deletes an empty directory 

mv [filename] [new_filename] > rename the file to a new filename 

cp -r [directoryname] [new_directoryname] > copy contents of first directory to the new directory 

ls -R Documents > see the contents of Documents w/o cding into it 

CTRL + r > search history: type a command ur looking for 

CTRL + c > stop current command 

CTRL + SHIFT + v > paste copied text into terminal 

ls -a > show all folders (including hidden)

uname -a > show system and kernel 

cat /etc/os-release > show OS release

lscpu > shows hardware info 

lsmem > memory info 

adduser, addgroup > adds a user/group 
```
